# TAS — Суть проекта

## Что это?

**TAS (Transmodal Anti-Spam)** — API для быстрого и точного обнаружения коммерческого спама в сообщениях.

## Проблема

Платформам нужна модерация спама, но:
- **Чистый LLM** — дорого ($0.01-0.05 за запрос) и медленно (500-2000ms)
- **Только правила** — много ложных срабатываний (FPR 20-30%)
- **Нет анализа картинок** — спамеры прячут рекламу в изображениях

## Решение

**Многослойная система** с умным fallback:

```
Сообщение
    ↓
1. Быстрые правила (200ms, $0) → 85% запросов решаются здесь
    ↓ (если не уверены)
2. Сигналы (URL риск, репутация) → дополнительный контекст
    ↓ (если всё ещё не уверены)
3. LLM fallback (700ms, $0.001) → только 15% запросов
    ↓ (если есть картинка)
4. Vision анализ (1-2s, $0.0001) → распознаёт текст в изображениях
    ↓
Финальный вердикт: spam/ham + причины
```

## Результат

- **Скорость**: 200ms (rules) / 700ms (с LLM)
- **Точность**: FPR < 5%, Recall 76%, Precision 95%
- **Стоимость**: 95% экономия vs чистый LLM
- **Transmodal**: Текст + изображения

## Логика работы

1. **Правила** проверяют паттерны (скидки, контакты, URL, промо-фразы)
2. **Если правила не уверены** → LLM анализирует контекст
3. **Если есть картинка** → Vision извлекает текст и проверяет
4. **Объединение результатов** → финальный score и причины

**Ключевая фишка**: 85% запросов обрабатываются правилами за 200ms без затрат. LLM вызывается только когда нужно.
