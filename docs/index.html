🔧 LLM Modes

TAS supports three modes for LLM usage:

Managed (Default)

Uses TAS-managed LLM credentials. No configuration needed.

curl -H "x-api-key: YOUR_KEY" ...

BYO (Bring Your Own)

Use your own LLM provider credentials for cost control and compliance.

curl -H "X-LLM-Mode: byo" \
     -H "X-LLM-Provider: openai" \
     -H "X-LLM-Key: sk-your-key" ...

Rules-Only

Fastest mode, no LLM costs. P95 ~200ms.

curl -H "X-LLM-Mode: rules_only" ...
